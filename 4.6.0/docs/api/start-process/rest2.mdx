---
title: Start Process
api: POST {baseUrl}/rtm/api/runtime/app/{appId}/build/{buildId}/process-name/{processDefinitionName}/start
description: This endpoint initiates a process in the application runtime environment using the specified `buildId` and `processDefinitionName`.
---


## Base URL

<ParamField path="baseUrl" type="string" example="https://admin.devmain.flowxai.dev">
  The base URL for all API requests. This remains constant across all requests and should be prepended to the endpoint path.
</ParamField>

## Path Parameters

<ParamField path="appId" type="string" example="dbe0ab87-539b-4015-962d-6e73226b9e58">
The ID of the application. 
</ParamField>

<ParamField path="buildId" type="string" example="7d5560be-3846-49fa-9c17-4059c54c7c5f"> 
The build ID for the runtime environment. 
</ParamField> 

<ParamField path="processDefinitionName" type="string" example="[DOCS]customer_onboarding"> 
The name of the process definition to start. 
</ParamField>
